# Bannières publicitaires / sponsors

Dépose ici les images (.jpg, .jpeg, .png, .gif, .webp) que tu veux voir
défiler sur l'écran neutre (avant/après concours), en plus du logo du club
et de l'horloge.

- L'écran les détecte automatiquement au démarrage du serveur, et se
  rafraîchit périodiquement -- pas besoin de relancer `run_server.py` si tu
  ajoutes une image en cours de concours (jusqu'à ~1 minute de délai).
- Aucun ordre particulier n'est garanti ; les images tournent l'une après
  l'autre avec l'horloge/logo entre chaque.
- Elles ne s'affichent jamais pendant le tir actif -- uniquement pendant
  les phases neutres, comme prévu dans les specs du projet.
