# Code de conduite

*[English version below](#code-of-conduct)*

FletchTime est un projet né d'un usage de club, ouvert à qui veut l'utiliser
ou y contribuer -- archers, responsables de club, développeurs curieux. Ce
qu'on attend, tout simplement :

- **Sois respectueux·se.** Les désaccords techniques sont normaux et sains
  -- ils portent sur des idées, jamais sur des personnes.
- **Sois patient·e.** Ce projet est porté par un club, pas une entreprise
  -- pas de réactivité garantie, pas de pression sur qui répond à une
  Issue ou une Pull Request.
- **Sois accueillant·e.** Une question qui semble évidente mérite une
  réponse aussi soignée qu'une question complexe -- tout le monde
  débute quelque part.
- **Reste sur le sujet du projet.** Harcèlement, propos discriminatoires,
  ou attaques personnelles ne sont pas tolérés, ici ou ailleurs en lien
  avec le projet (Issues, Pull Requests, discussions).

## Signaler un souci

Si le comportement de quelqu'un pose problème, contacte le mainteneur du
projet directement via une
[Issue GitHub](https://github.com/MrFanghoDev/fletchtime/issues) marquée
comme privée si possible, ou par le contact indiqué sur son profil GitHub.
Chaque signalement sera pris au sérieux et traité avec discrétion.

## Portée

Ce texte s'applique à tous les espaces du projet (Issues, Pull Requests,
discussions) et à toute interaction publique s'en réclamant directement.

# Code of Conduct

FletchTime started from a club's real-world use, open to anyone who wants
to use it or contribute -- archers, club officials, curious developers.
What's expected, simply put:

- **Be respectful.** Technical disagreements are normal and healthy --
  they're about ideas, never about people.
- **Be patient.** This project is run by a club, not a company -- no
  guaranteed responsiveness, no pressure on whoever answers an Issue or
  Pull Request.
- **Be welcoming.** A question that seems obvious deserves as thoughtful
  an answer as a complex one -- everyone starts somewhere.
- **Stay on topic.** Harassment, discriminatory remarks, or personal
  attacks aren't tolerated, here or elsewhere in connection with the
  project (Issues, Pull Requests, discussions).

## Reporting a concern

If someone's behavior is a problem, contact the project maintainer
directly via a
[GitHub Issue](https://github.com/MrFanghoDev/fletchtime/issues) marked
private if possible, or through the contact listed on their GitHub
profile. Every report will be taken seriously and handled with discretion.

## Scope

This applies to all project spaces (Issues, Pull Requests, discussions)
and to any public interaction directly referencing the project.
