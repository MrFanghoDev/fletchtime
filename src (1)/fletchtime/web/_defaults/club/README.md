# Logo du club

Dépose ici le logo de ton club (formats `.jpg`, `.png` ou `.svg`) pour
qu'il apparaisse sur l'écran neutre entre les phases de tir.

FletchTime cherche automatiquement, dans cet ordre, un fichier nommé
exactement :

1. `logo.jpg`
2. `logo.png`
3. `logo.svg`

Tant que ce dossier ne contient aucun de ces trois noms, l'écran neutre
affiche le logo FletchTime à la place.
