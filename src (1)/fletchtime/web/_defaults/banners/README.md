# Bannières publicitaires / sponsors

Dépose ici les images de tes sponsors (`.jpg`, `.png`, `.gif`, `.webp` --
n'importe quel nom de fichier) pour qu'elles défilent automatiquement sur
l'écran neutre, en alternance avec le logo du club.

Elles ne s'affichent jamais pendant le tir actif -- uniquement pendant les
phases neutres (avant/après le concours, entre deux matchs).
